**1. Overview**
Clay's Piknik is a responsive landing page website created for a small food business based in Bali. The website showcases homemade comfort food products, introduces the founder, highlights the brand's unique values, and provides customers with simple ordering information. The website was designed with a warm, inviting visual style inspired by a picnic, creating a friendly and memorable user experience.

**2. Features and Sections**
This website is constructed by several section that has different roles to provide the potential customers with information about Clay’s Piknik. These features include : 
a. Responsive Design
   The website adapts to the screen size to provide better visualization
b. Product Showcase
   Displays Clay's Piknik menu, including:
    - Picnic Roll
    - Mac n Cheese
    - Spaghetti Brulee
    - Risoles
c. Founder Introduction
   Introduces the owner and shares the story behind the business.
d. Why Choose Us Section
   Highlights the key strengths of the brand:
    - Freshly Made
    - High Quality Ingredients
    - Memorable Flavours
    - Perfect For Sharing
e. Delivery
   Provides delivery information to acknowledge the customer about the delivery system. 
g. Call-to-Action (CTA)
   Encourages visitors to place an order through WhatsApp.
h. Interactive User Experience
   Includes:
  - Smooth scrolling navigation
  - Scroll-triggered animations
  - Hover effects and transitions

**3. Code Language Used**
This website utilizes several coding languages to operate different functions to support the website functionality and aesthetic, including : 
a. HTML5
   Construct website structure and website content.
b. CSS
   Adding a design attribute to an element of HTML.
d. JavaScript
   Operate the logic behind the website to support website functionality.
e. Google Fonts (Poppins)
   Import the font Poppins as the main font used on this website.

**5. Project Structure**
This project consists of 2 files of HTML and CSS. There’s also an additional folder of assets that provides supporting content for the website, including icons and pictures. 

**6. Design Goals**
The website was designed with the following objectives:
  - Create a clean and modern user interface.
  - Establish a warm and friendly brand identity.
  - Showcase products through an appealing visual presentation.
  - Encourage customers to place an order through call-to-action elements.

**7. How to Run the Project**
Download all the files (index.html, style.css, assets folder)
1. Ensure all asset files are located inside the assets folder.
2. Make sure every element of the file is located in one folder.
3. Open index.html in any modern web browser.
No additional installation or dependencies are required.
